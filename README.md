# G R O U P

Group website for ardency businesses ( `funding`: `ardency.vc`) and ( `tech`: `ardency.io`, `ardency.sh`)

## Features

- Astro v2.x
- Nodejs v19.x
- Bulma v0.9.x
- Alpine v3.x

## Usage

### Install Depedencies

```sh
pnpm i
```

### Run in dev mode

```sh
pnpm dev
```

### build source

```sh
pnpm build
```
